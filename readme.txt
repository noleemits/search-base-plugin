A custom search plugin with pretty permalinks and hierarchical taxonomy handling.

== Description ==

Custom Search Base is a powerful plugin that provides custom search functionality with category and metro filters. This plugin supports pretty permalinks and hierarchical taxonomy handling.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/custom-search-base` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the shortcodes `[lawyer_search_form]` and `[metro_filter_form]` to display the search forms on your site.
